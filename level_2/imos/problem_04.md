# いもす法 問題4：等差数列の加算

## 問題

長さNの配列（初期値は全て0）に対して、Q個の操作を行う。
各操作で区間 [L, R] に等差数列 1, 2, 3, ... を加算する。
（A[L] += 1, A[L+1] += 2, ..., A[R] += R-L+1）
全操作後の配列を出力せよ。（1-indexed）

## 入力

```
N Q
L1 R1
L2 R2
...
LQ RQ
```

## 出力

N個の整数

## 入力例

```
5 2
1 3
2 5
```

## 出力例

```
1 3 4 3 4
```

## 補足

操作1: [1,2,3,0,0]、操作2: [0,1,2,3,4] → 合計 [1,3,5,3,4]... 
実際は [1,2,3,0,0] + [0,1,2,3,4] = [1,3,5,3,4]
