# 二分探索（答えで探索）問題3：仕事の割り当て

## 問題

N個の仕事があり、仕事iの所要時間はT[i]である。
K人の作業者に仕事を順番に割り当てる（仕事の順序は変えられない）。
各作業者の仕事時間の最大値を最小化せよ。

## 入力

```
N K
T1 T2 ... TN
```

## 出力

最大仕事時間の最小値

## 入力例

```
5 2
1 2 3 4 5
```

## 出力例

```
9
```

## 補足

[1,2,3,4] と [5] に分けると max(10, 5) = 10
[1,2,3] と [4,5] に分けると max(6, 9) = 9（最小）
