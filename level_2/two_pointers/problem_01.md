# しゃくとり法 問題1：和がS以上の最短区間

## 問題

長さNの正整数配列Aが与えられる。
和がS以上になる連続部分配列の最短の長さを求めよ。
存在しない場合は 0 を出力せよ。

## 入力

```
N S
A1 A2 ... AN
```

## 出力

最短の長さ（存在しなければ 0）

## 入力例

```
6 15
2 3 1 2 4 3
```

## 出力例

```
2
```

## 補足

[4, 3] の和が 7... ではなく [2,4,3]=9, [1,2,4,3]=10... 
実際は [4,3]=7 < 15、[2,4,3]=9 < 15、[1,2,4,3]=10 < 15、[3,1,2,4,3]=13 < 15、[2,3,1,2,4,3]=15 で長さ6
