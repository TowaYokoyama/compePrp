# 制約→計算量逆算 問題5 メモ

## 制約と計算量の目安

- N ≤ 2 × 10^5 → O(N^2) は TLE
- O(N log N) または O(N) が必要

## この問題

- Counter で頻度を数えて O(N)
- ソート + 二分探索でも O(N log N)
- 2重ループは 4 × 10^10 回で TLE

## 制約から解法を逆算

| 制約 | 計算量 | 解法例 |
|------|--------|--------|
| N ≤ 10 | O(N!) | 順列全探索 |
| N ≤ 20 | O(2^N) | bit全探索 |
| N ≤ 500 | O(N^3) | 3重ループ |
| N ≤ 10^4 | O(N^2) | 2重ループ |
| N ≤ 10^6 | O(N log N) | ソート、二分探索 |
| N ≤ 10^7 | O(N) | 線形探索 |
