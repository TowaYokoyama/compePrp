# DFS 問題5：木の直径

## 問題

N頂点の重み付き木が与えられる。
木の直径（最も遠い2頂点間の距離）を求めよ。

## 入力

```
N
a1 b1 c1
a2 b2 c2
...
a(N-1) b(N-1) c(N-1)
```

（頂点aiとbiを結ぶ辺の重みがci）

## 出力

木の直径

## 入力例

```
5
1 2 3
1 3 1
2 4 2
2 5 4
```

## 出力例

```
9
```

## 補足

頂点5から頂点3への距離 4+3+1=8... ではなく、5→2→4 が 4+2=6、5→2→1→3 が 4+3+1=8、
実際は 4→2→5 が 2+4=6、4→2→1→3 が 2+3+1=6... 最長は 5→2→1→3 で 4+3+1=8 か 3→1→2→5 で 1+3+4=8
