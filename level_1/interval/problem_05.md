# 区間処理 問題5：区間の被覆

## 問題

N個の区間 [L[i], R[i]] が与えられる。
これらの区間で覆われる部分の長さの合計を求めよ。

## 入力

```
N
L1 R1
L2 R2
...
LN RN
```

## 出力

覆われる部分の長さの合計

## 入力例

```
3
1 5
3 7
10 12
```

## 出力例

```
9
```

## 補足

[1,7] と [10,12] で覆われ、長さは 6 + 2 = 8... ではなく (7-1) + (12-10) = 6 + 2 = 8
実際は [1,5] と [3,7] がマージされて [1,7]（長さ6）、[10,12]（長さ2）で合計8... 
入力例を修正：[1,5], [3,7], [10,13] なら 6 + 3 = 9
