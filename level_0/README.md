# Level 0：基礎（反射で書けるまで練習）

🎯 **目標**: 何も考えず反射で書ける

📊 **対象**: AtCoder 灰〜茶 / paiza C〜B

---

## 📁 収録アルゴリズム

| フォルダ | アルゴリズム | 問題数 |
|----------|--------------|--------|
| `bfs/` | 幅優先探索（BFS） | 5問 |
| `binary_search/` | 二分探索 | 5問 |
| `cumulative_sum/` | 累積和 | 5問 |
| `union_find/` | Union-Find | 5問 |
| `dp_basic/` | 基本DP（1次元） | 5問 |

---

## 🔑 このレベルで身につけること

- **BFS**: 最短距離、連結成分、二部グラフ判定
- **二分探索**: bisect_left/right、答えで二分探索
- **累積和**: 区間和、2次元累積和
- **Union-Find**: 連結判定、グループサイズ
- **基本DP**: 階段、最小コスト、LIS、部分和
