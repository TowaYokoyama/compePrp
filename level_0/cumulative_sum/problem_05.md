# 累積和 問題5：偶奇で分ける累積和

## 問題

長さNの配列Aが与えられる。
偶数番目の要素の和から奇数番目の要素の和を引いた値が0になる区間 [L, R] の個数を求めよ。（1-indexed）

## 入力

```
N
A1 A2 ... AN
```

## 出力

条件を満たす区間の個数

## 入力例

```
5
1 2 3 4 5
```

## 出力例

```
2
```

## 補足

区間 [1,2]: A1=1, A2=2 → 2-1=1 ≠ 0
区間 [1,4]: A1+A3=4, A2+A4=6 → 6-4=2 ≠ 0
区間 [2,3]: A2=2, A3=3 → 2-3=-1 ≠ 0
区間 [1,1]: 0-1=-1 ≠ 0
...
