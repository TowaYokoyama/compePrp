# 基本DP 問題2：最小コストで移動

## 問題

N個のマスが一列に並んでいる。マスiにはコストC[i]がかかる。
マス1からスタートし、1マスまたは2マス進めるとき、マスNに到達する最小コストを求めよ。
（マス1のコストも含む）

## 入力

```
N
C1 C2 ... CN
```

## 出力

最小コスト

## 入力例

```
5
1 2 1 3 1
```

## 出力例

```
4
```

## 補足

マス1→3→5 と進むと 1+1+1=3... ではなく、1→2→4→5 で 1+2+3+1=7... 
実際は 1→3→5 で 1+1+1=3 が最小（訂正：マス1,3,5のコスト合計）
