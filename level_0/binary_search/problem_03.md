# 二分探索 問題3：条件を満たす最小値

## 問題

N個の荷物があり、それぞれの重さは W1, W2, ..., WN である。
トラックの最大積載量を X としたとき、全ての荷物を運ぶのに必要な最小トラック台数を求める関数がある。
K台以下で全ての荷物を運べる最小の積載量Xを求めよ。
（荷物は分割できず、順番通りに積む）

## 入力

```
N K
W1 W2 ... WN
```

## 出力

必要な最小積載量

## 入力例

```
5 3
1 2 3 4 5
```

## 出力例

```
6
```

## 補足

積載量6なら [1,2,3], [4], [5] の3台で運べる
