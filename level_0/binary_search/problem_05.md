# 二分探索 問題5：K番目に小さい値

## 問題

長さNの配列Aと長さMの配列Bが与えられる。
A[i] + B[j] (1 <= i <= N, 1 <= j <= M) の全ての組み合わせのうち、
K番目に小さい値を求めよ。

## 入力

```
N M K
A1 A2 ... AN
B1 B2 ... BM
```

## 出力

K番目に小さい値

## 入力例

```
3 3 5
1 3 5
2 4 6
```

## 出力例

```
7
```

## 補足

全組み合わせ: 3,5,7,5,7,9,7,9,11 → ソート: 3,5,5,7,7,7,9,9,11 → 5番目は7
