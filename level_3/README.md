# Level 3：アルゴリズム（本格的なアルゴリズム）

🎯 **目標**: グラフ・木のアルゴリズムを使いこなす

📊 **対象**: AtCoder 緑〜水 / paiza A

---

## 📁 収録アルゴリズム

| フォルダ | アルゴリズム | 問題数 |
|----------|--------------|--------|
| `dijkstra/` | ダイクストラ法 | 5問 |
| `bfs_01/` | 0-1 BFS | 5問 |
| `bellman_ford/` | Bellman-Ford | 1問 |
| `warshall_floyd/` | ワーシャルフロイド | 1問 |
| `tree_dfs/` | 木のDFS | 1問 |
| `tree_dp/` | 木DP | 1問 |
| `tree_diameter/` | 木の直径 | 2問 |
| `lca/` | LCA（最近共通祖先） | 1問 |
| `interval_dp/` | 区間DP | 1問 |

---

## 🔑 このレベルで身につけること

- **ダイクストラ**: 重み付き最短経路、経路復元
- **0-1 BFS**: 辺の重みが0か1のグラフ
- **Bellman-Ford**: 負の辺がある最短経路
- **ワーシャルフロイド**: 全点対最短経路
- **木のDFS**: 部分木サイズ、親子関係
- **木DP**: 独立集合、塗り分け
- **LCA**: ダブリングによる最近共通祖先
