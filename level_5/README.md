# Level 5：データ構造（上級入口）

🎯 **目標**: 高度なデータ構造を使いこなす

📊 **対象**: AtCoder 水〜青 / paiza S

---

## 📁 収録アルゴリズム

| フォルダ | アルゴリズム | 問題数 |
|----------|--------------|--------|
| `segment_tree/` | セグメント木 | 1問 |
| `fenwick_tree/` | Fenwick Tree（BIT） | 1問 |
| `priority_queue/` | Priority Queue（heap） | 1問 |

---

## 🔑 このレベルで身につけること

- **セグメント木**: 1点更新・区間クエリ
- **Fenwick Tree**: 1点加算・区間和（セグ木より軽量）
- **Priority Queue**: K番目の値、ダイクストラ

---

## 📊 使い分け

| データ構造 | 更新 | クエリ | 用途 |
|------------|------|--------|------|
| 累積和 | - | O(1) | 静的な区間和 |
| Fenwick Tree | O(log N) | O(log N) | 動的な区間和 |
| セグメント木 | O(log N) | O(log N) | 区間min/max/和など |
