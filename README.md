# 競技プログラミング学習教材

AtCoder / paiza / Codeforces 向けの学習教材です。

## 📁 フォルダ構成

```
├── level_0/    # 基礎（反射で書けるまで練習）
├── level_1/    # 重要（問題文読んで即これ）
├── level_2/    # 中級への橋渡し（解法選択力）
├── level_3/    # アルゴリズム（本格的なアルゴリズム）
├── level_4/    # DPの壁（避けては通れない）
├── level_5/    # データ構造（上級入口）
└── tips/       # テクニック集
```

## 🎯 レベル別目安

| レベル | AtCoder | paiza | 内容 |
|--------|---------|-------|------|
| 0 | 灰〜茶 | C〜B | 基礎アルゴリズム |
| 1 | 茶〜緑 | B | 典型パターン |
| 2 | 緑 | B〜A | 中級テクニック |
| 3 | 緑〜水 | A | グラフ・木 |
| 4 | 水 | A〜S | 発展DP |
| 5 | 水〜青 | S | データ構造 |

## 📝 各フォルダの構成

```
algorithm_name/
├── problem_01.md    # 問題文
├── solution_01.py   # 正解コード
├── memo_01.md       # ポイント・着眼点・よくあるミス
├── problem_02.md
├── solution_02.py
├── memo_02.md
...
```

## 🚀 学習の進め方

1. `problem_XX.md` を読んで問題を解く
2. 解けなければ `memo_XX.md` のヒントを見る
3. `solution_XX.py` で答え合わせ
4. 「何も考えず反射で書ける」まで繰り返す
